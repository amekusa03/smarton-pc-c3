## 確定仕様（2026年5月4日）

| 機能 | 手段 |
|------|------|
| 起動 | フォトカプラ → 電源ボタン短押し |
| フリーズ回復 | フォトカプラ → 電源ボタン長押し |
| 死活監視 | Ping（ホスト名指定） |
| 通知 | Google Home |

- WoL は信頼性の問題により採用しない
- IPアドレス変動対策はPC側でIP固定 or ホスト名（mDNS）で対応
- シャットダウンはソフト（SSH等）で対応、アクティビティ図のメインフローには含めない
- ネット未接続時の監視失敗は諦める（スマートホーム自体が機能しないため）
